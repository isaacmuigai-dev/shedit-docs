# AmiPal Landing Page 🚀

Welcome to the official landing page for **AmiPal**, the premium AI-powered calorie tracking app.

## ✨ Features

- **AI Food Recognition**: Interactive scanning simulation.
- **Modern Design**: Dark mode, glassmorphism, and smooth animations.
- **Framer Motion**: Scroll-triggered reveals and micro-interactions.
- **Tailwind CSS**: Fully responsive and optimized styling.
- **Mobile First**: Designed for the modern mobile user.

## 🛠️ Tech Stack

- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Locally**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## 🌐 Deployment (GitHub Pages)

The project is configured to be hosted on GitHub Pages. 

1. Push your changes to the `main` branch.
2. Ensure `vite.config.js` has `base: './'`.
3. Use a GitHub Action or the `gh-pages` package to deploy the `dist` folder.
